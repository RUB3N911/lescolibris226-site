# Les Colibris 226

Site officiel de l’association culturelle Les Colibris du Morne-Vert.

## Signature

Le battement culturel du Morne-Vert.

## Objectif

Créer une plateforme culturelle immersive pour présenter l’association, son histoire, ses événements, sa galerie, ses partenaires et permettre aux visiteurs de rejoindre l’aventure.

## Stack prévue

- React
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React
- GitHub
- Vercel

## Pages prévues

- Accueil
- Histoire
- Événements
- Galerie
- Rejoindre

- Contact
